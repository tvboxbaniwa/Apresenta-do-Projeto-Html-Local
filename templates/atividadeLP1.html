Exercícios de Python

Execute os códigos no seu terminal e informe abaixo a saída gerada.

Exercício de Python 1: tipos de dados

Dada a lista:

itens = [123, "texto", 3.14, True, None]

Escreva um programa que imprime o tipo de cada item da lista usando type(). Exemplo de saída esperada:

<class 'int'>
<class 'str'>
...

Exercício de Python 2: soma de dois números

Escreva um programa que leia dois números (2 e 3) e imprima a soma.

Exercício de Python 3: maior e menor palavra

Partindo de uma lista de palavras:

palavras = ["python", "asimov", "código", "web", "programação"]

Escreva um programa que imprima a maior e a menor palavra da lista (baseado no número de caracteres).

Formato da saída esperada (exatamente):

Maior: programação
Menor: web
